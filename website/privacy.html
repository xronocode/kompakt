Privacy Policy

Last updated: April 6, 2026

pdf-kompakt does not collect, store, transmit, or share any user data. Period.

Overview

pdf-kompakt is a free, open-source PDF compression tool available as a desktop CLI application and a Chrome browser extension. Both versions process files entirely on your device. No data ever leaves your computer.

Desktop Application

The desktop CLI application (pdf-kompakt):

  • Runs 100% locally on your machine
  • Does not connect to any server or external service
  • Does not collect usage data, analytics, or telemetry
  • Does not require an account or registration
  • Does not access any files other than the PDF you choose to compress
  • Works fully offline with no internet connection required

Chrome Extension

The Chrome browser extension (pdf-kompakt):

  • Processes all PDF files locally in your browser using JavaScript
  • Does not upload files to any server
  • Does not collect, store, or transmit any personal information
  • Does not use cookies, local storage for tracking, or any analytics
  • Does not access your browsing history, bookmarks, or other browser data
  • Does not require any special permissions beyond file access for compression
  • Works fully offline after installation

Landing Page (this website)

The website at kompakt.xronocode.com:

  • Is a static HTML page hosted on Cloudflare Pages
  • Does not use cookies for tracking
  • Does not collect personal information
  • Stores only your language preference in localStorage (never transmitted)
  • Uses a Ko-fi widget for voluntary donations (Ko-fi's own privacy policy applies to donations)

Third-Party Services

  • Cloudflare Pages — hosts this website. Cloudflare may collect standard web server logs (IP address, user agent). See Cloudflare's Privacy Policy.
  • Ko-fi — used for voluntary donations. If you choose to donate, Ko-fi's privacy policy applies. See Ko-fi's Privacy Policy.
  • GitHub — hosts the source code and binary releases. See GitHub's Privacy Statement.

Data Retention

Since we do not collect any data, there is nothing to retain or delete.

Children's Privacy

pdf-kompakt does not collect any information from anyone, including children under 13.

Changes to This Policy

If this policy is updated, the changes will be posted on this page with an updated date. Since we collect no data, meaningful changes are unlikely.
